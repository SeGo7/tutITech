Политика конфиденциальности

Дата вступления в силу: 29.04.2026

Настоящая Политика конфиденциальности определяет порядок обработки и защиты персональных данных пользователей сайта ITtutor.

1. Какие данные мы собираем

  • имя;
  • номер телефона;
  • контакт для связи, если пользователь указывает его в форме;
  • класс, предмет, экзамен и цель обращения;
  • иные сведения, которые пользователь указывает в форме заявки добровольно.

2. Цели обработки персональных данных

  • обратная связь с пользователем по оставленной заявке;
  • консультация по услугам подготовки к экзаменам;
  • согласование расписания и условий занятий.

3. Правовые основания обработки

Обработка персональных данных осуществляется на основании согласия пользователя, выраженного при отправке заявки на сайте.

4. Где и как обрабатываются данные

Заявка на сайте собирается через встроенную форму сервиса Яндекс Формы. При отправке формы данные передаются в сервис Яндекс Формы и становятся доступны ответственному лицу ITtutor для обработки заявки, обратной связи и согласования занятий.

5. Передача данных третьим лицам

Для работы формы персональные данные передаются сервису Яндекс Формы в объёме, необходимом для приёма и хранения заявки. Иная передача третьим лицам не осуществляется, за исключением случаев, предусмотренных законодательством Российской Федерации.

6. Срок хранения данных

Персональные данные хранятся не дольше, чем это необходимо для достижения целей обработки, либо до отзыва согласия пользователем.

7. Права пользователя

  • получить информацию о своих персональных данных;
  • требовать уточнения, блокирования или удаления персональных данных;
  • отозвать согласие на обработку персональных данных.

8. Контакты по вопросам персональных данных

По вопросам обработки персональных данных пользователь может обратиться через контактные способы связи, указанные на сайте ITtutor.
